## Features
- Book like behaviour achieved using the Turn.js library 📕
- Custom Font made suing Calligraphr 
- Custom cutting matt background made with CSS 
- Optimized performance and size for both mobile and desktop devices 💻
- On mobile devices, Turn,js will not be initialised and instead the page will enable scroll 📱
